PaperFrame

Introduction

What is PaperFrame?

PaperFrame is a material design framework based on vanilla HTML, CSS and JS. PaperFrame is currently in a pre-alpha stage. PaperFrame's goal is to be super fast and super light weight.

This framework is made with the material design guidelines not only as guidelines, but as rules. These pages serve as demo, documentation and instruction pages

Getting set up

Paperframe is available in standard CSS and SCSS. It is recommended to use the SCSS variant in order to have a finer control on all elements and details, including global colour scheme.

CSS

To get started with Paperframe using CSS, download the CSS variant into your project, then simply use the code below to start your project. 
              
<!DOCTYPE html>
<html id="paperframe">
<head>
  <script src="https://code.jquery.com/jquery-2.1.4.min.js"></script>
  <link rel="stylesheet" href="paperframe/paperframe.min.css">
  <script src="paperframe/paperframe.min.js"></script>
  <meta name="viewport" content="width=device-width, initial-scale=1.0 user-scalable=0" />
</head>
<body>
  <!--Content goes here -->
</body>
</html>

SCSS

Assuming that you already know how to compile SCSS, simply add the following code at the top of your main SCSS file if your main SCSS file is in the root directory of your project.

              
@import 'paperframe/paperframe';
