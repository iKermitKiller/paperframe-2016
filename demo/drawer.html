Drawer

An introduction to the drawer

When there is insufficient space to support tabs, side navigation is a good alternative. Side navigation can display many navigation targets at once. A drawer remains hidden until invoked by the user.

Apps with a single "home" should list the most frequently accessed destinations at the top of side navigation.

Side navigation may appear either with or without a navigation drawer.

Appropriate for these hierarchies:

  • Lateral navigation
  • Parents with siblings or peers

Recommended for:

  • Apps with many top-level views
  • Enabling quick navigation between unrelated views
  • Deep navigation structures
  • Reducing visibility of infrequent destinations

Code

              
<div id="drawer">
  <div class="header">
    <div class="user">
      <div class="avatar"></div>
      <div class="name">First Last</div>
      <div class="email">email@email.com</div>
    </div>
  </div>
  <ul>
    <a href="#">
      <li>
        <i class="material-icons">info</i>
        <span>Item</span>
      </li>
    </a>
    <hr>
    <h2>Section</h2>
    <a href="#">
      <li>
        <i class="material-icons">info</i>
        <span>Item</span>
      </li>
    </a>
  </ul>
</div>
<div id="scrim" onclick="toggleDrawer()"></div>

This code alone has no toggle to activate the drawer manually when the drawer is off canvas, the following code will activate the drawer

              
<button id="hamburger" onclick="toggleDrawer()">Toggle drawer</button>

You can apply the ID and the onclick attribute to any element.

Extra

The drawer will always be visible when viewing on desktop or on a large tablet. On small tablets or phones, the drawer will be off canvas. When the drawer is off canvas, to reveal the drawer you must activate it.

Item icons

You can choose to either have an icon for each drawer item or not have one. To not have a drawer icon, remove the following code from the drawer item you don't want an icon on:

                
<i class="material-icons">info</i>

To add an icon to a drawer item, simply add the previous code before the span tag. The full list of usable icons are available here
