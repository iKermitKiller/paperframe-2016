Snackbars and toasts

An introduction to snackbars and toasts

Snackbars

Snackbars provide lightweight feedback about an operation by showing a brief message at the bottom of the screen. Snackbars can contain an action.

The snackbar action will use the secondary colour defined in your SCSS colours file, located in /scss/import/colours.scss

Toasts

Toasts are similar to snackbars but do not contain actions and cannot be dismissed manually.

Code

Snackbars

The snackbar is called using the toggleSnackbar() function. The snackbar must manually be added to the page, beneath everything in the HTML document but above the closing body tag.

HTML

              
<div id="snackbar" class="hidden">
  <div class="content">
    This is a snackbar
    <div class="actions">
      <button flat>action</button>
    </div>
  </div>
</div>

Toasts

Toats are called using the toggleToast() function. Toasts must manually be added to the page, beneath everything in the HTML document but above the closing body tag.

HTML

              
<span id="toast">This is a toast</span>
This is a snackbar
This is a toast
