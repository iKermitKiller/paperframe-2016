Colours

An introduction to colours

All of the colours available on the official material design guidelines provided by Google are available within Paperframe, including the variants of each colour. Currently these colours are only available when using Paperframe with SCSS.

Code

Mixins (SCSS)

HTML

            
<div id="test"></div>
<div id="test2"></div>

SCSS

            
#test{
  @include indigo(700);
}
#test2{
  @include pink(400);
}
